# KT CS AI 에이전트 상담 시스템

## 📋 프로젝트 개요
KT CS에서 AI 에이전트가 먼저 고객과 상담한 후, 그 결과를 실제 상담사에게 전달하는 시스템입니다. 
상담사는 AI가 이미 처리한 대화 내용과 분석 결과를 보고 후속 처리를 진행합니다.

## 🤖 시스템 구조

### 1단계: AI 상담 (고객 ↔ AI 에이전트)
- 고객이 KT에 전화
- AI 상담사가 자동 응답 (STT/TTS 사용)
- 요금제 상담 진행
- 필요시 실제 상담사에게 전달

### 2단계: 상담사 후속 처리 (현재 화면)
- AI 상담 대화록 확인
- AI가 분석한 요약 및 추천 요금제 검토
- 고객 정보 확인
- 요금제 변경 등 실제 처리 진행

## 🎯 주요 기능

### 1. AI 상담 대화록
- 고객과 AI 상담사의 전체 대화 내용
- 타임스탬프별 정리
- 대화 시간 및 메시지 수 표시
- AI 상담 요약 자동 생성

### 2. AI 요약 및 추천 요금제
- 고객 상황 자동 분석
- 최적 요금제 추천 (가족결합 할인 포함)
- 업세일링 기회 제시
- 절약형 대안 제공

### 3. 상세 고객 정보
- 기본 정보: 성별, 나이, 생년월일
- KT 가입 정보: 가입일, 이용 기간
- 현재 요금제 및 서비스
- 가족 결합 현황
- VIP 등급 표시

### 4. 요금제 검색
- 실시간 요금제 검색
- 카테고리별 필터링
- 관련도 기반 정렬

## 🛠️ 기술 스택

### Frontend
- **React 18** - UI 프레임워크
- **Styled Components** - CSS-in-JS 스타일링
- **Framer Motion** - 애니메이션
- **React Icons** - 아이콘

### AI/Backend (연동 예정)
- **LangChain** - AI 오케스트레이션
- **STT/TTS** - 음성 인식/합성
- **RAG + VectorDB** - 요금제 정보 검색
- **FastAPI** - 백엔드 서버

## 🎨 UI/UX 특징

### 레이아웃
- **3단 그리드 구조**
  - 왼쪽: 고객 정보 + 요금제 검색
  - 중앙: AI 상담 대화록
  - 오른쪽: AI 요약 및 추천 요금제

### 디자인 특징
- AI 상담 완료 상태 실시간 표시
- 대화 로그 시간순 정렬
- 추천 요금제 우선순위 표시 (최적/업그레이드/절약형)
- VIP 고객 시각적 구분

## 🔄 상담 프로세스

1. **AI 상담 단계** (자동화)
   ```
   고객 전화 → AI 응답 → 요구사항 파악 → 요금제 추천 → 상담사 전달
   ```

2. **상담사 처리 단계** (현재 화면)
   ```
   대화록 검토 → AI 분석 확인 → 추천 요금제 검토 → 실제 처리
   ```
